\emph{
	The advancement of Unmanned Aerial Vehicle (UAV) technology has created new opportunities for Search and Rescue (SAR) operations. In Indonesia, where natural disasters occur frequently, there is an urgent need for systems capable of assisting victim detection efficiently in areas that are difficult for humans to access. This study aims to design an autonomous system for the Holybro S500 drone capable of detecting disaster victims automatically using a Raspberry Pi 5 and a webcam. The system integrates a Pixhawk 6C Flight Controller as the main navigation controller and a Raspberry Pi 5 as a companion computer responsible for autonomous control via the MAVLink protocol (MAVSDK Python) and human detection algorithms. Based on evaluation results, a fine-tuned, standard YOLOv8n model in ONNX format (320x320 resolution) was selected over a pose estimation model, as it proved to be more consistent and stable in maintaining bounding boxes in outdoor environments. This research also implements an asynchronous multi-threaded software architecture to prevent control latency, as well as an isolated electrical system design using a 5A UBEC regulator which effectively prevents voltage sag on the Raspberry Pi 5 during drone maneuvers. Overall testing results indicate that.} %the system prototype successfully integrates visual perception modules, state machine-based autonomous navigation, and real-time remote monitoring, serving as a solid foundation for developing efficient, responsive, and reliable autonomous SAR drones.}
